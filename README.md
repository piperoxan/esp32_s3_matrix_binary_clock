# esp32_s3_matrix_binary_clock
适用于微雪 ESP32-S3-Matrix 的二进制时钟 小程序, 使用arduino 编写  
参考链接: https://www.waveshare.net/wiki/ESP32-S3-Matrix  
环境配置:见参考链接中arduino 环境搭建部分   

功能简要说明:
一个联网二进制时钟, 添加重力感应旋转功能,
